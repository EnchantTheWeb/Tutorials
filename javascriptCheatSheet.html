üìÑ Document Structure Elements

<!DOCTYPE html>
Tells the browser this is an HTML5 document. Always put this at the very top!
<!DOCTYPE html>
<html>
The root element that wraps all content on the page
<html lang="en">
  All your content goes here
</html>
<head>
Contains information about the page that doesn't show up on the page itself (like the title in the browser tab)
<head>
  <title>My Website</title>
  <meta charset="UTF-8">
</head>
<body>
Contains all the visible content of your webpage
<body>
  <h1>Welcome to my site!</h1>
  <p>This is visible content</p>
</body>

üíª JavaScript Basics

<script>
Contains JavaScript code that makes your webpage interactive
<script>
  alert("Hello World!");
  console.log("This appears in the browser console");
</script>
Variables in JavaScript
Store data in variables using let, const, or var
let name = "John";
const age = 25;
var city = "New York";
Use 'let' for variables that can change, 'const' for constants that won't change.
Functions in JavaScript
Functions are reusable blocks of code
function greetUser(name) {
  return "Hello, " + name + "!";
}

// Call the function
let message = greetUser("Alice");
DOM Manipulation
Change HTML elements with JavaScript
// Get an element by its ID
document.getElementById("myElement")

// Change text content
document.getElementById("demo").innerHTML = "New text!";

// Change styles
document.getElementById("demo").style.color = "red";

üí° Beginner Tips

Essential Tips for New Coders:

  • Always close your tags! Every opening tag needs a closing tag (except self-closing tags like <br> and <img>)
  • Indent your code to make it readable and see the structure clearly
  • Use meaningful names for IDs and classes (like "navigation" instead of "div1")
  • Test frequently - save your file and refresh the browser often to see changes
  • Use the browser's developer tools (press F12) to inspect elements and debug
  • Comments are your friend! Use <!-- comments --> in HTML and // comments in JavaScript
