Welcome to HTML! HTML (HyperText Markup Language) is the foundation of all websites. It uses "tags" (words in angle brackets like <tag>) to structure and format content. Most tags come in pairs: an opening tag <tag> and a closing tag </tag>.

üìã Document Structure Tags

<!DOCTYPE html>
Tells the browser this is an HTML5 document. Always put this at the very top of your HTML file.
Example: <!DOCTYPE html>
<html></html>
The root element that wraps all content on the page. Everything goes inside this.
Example: <html lang="en">...</html>
<head></head>
Contains metadata (information about the page) that doesn't show on the webpage itself.
Example: <head><title>My Page</title></head>
<body></body>
Contains all the visible content of your webpage - text, images, links, etc.
Example: <body><h1>Welcome!</h1></body>

üìù Meta Information Tags

<title></title>
Sets the title that appears in the browser tab and search results.
Example: <title>My Amazing Website</title>
<meta>
Provides metadata like character encoding, description, and viewport settings. Self-closing tag.
Examples:
<meta charset="UTF-8"> (character encoding)
<meta name="description" content="My website description">

üì∞ Heading Tags

<h1></h1> to <h6></h6>
Create headings of different sizes. h1 is the largest/most important, h6 is the smallest.
Examples:
<h1>Main Title</h1>
<h2>Section Title</h2>
<h3>Subsection</h3>

üìÑ Text Content Tags

<p></p>
Creates a paragraph of text. Most text content should go in paragraphs.
Example: <p>This is a paragraph of text.</p>
<br>
Creates a line break (like pressing Enter). Self-closing tag. Use sparingly!
Example: First line<br>Second line
<hr>
Creates a horizontal rule (line) to separate content. Self-closing tag.
Example: <hr>

‚ú® Text Formatting Tags

<strong></strong>
Makes text bold and indicates it's important.
Example: <strong>This is important!</strong>
<em></em>
Makes text italic and emphasizes it.
Example: <em>This is emphasized</em>
<b></b>
Makes text bold (visual only, no special meaning).
Example: <b>Bold text</b>
<i></i>
Makes text italic (visual only, no special meaning).
Example: <i>Italic text</i>
<u></u>
Underlines text. Be careful - users might think it's a link!
Example: <u>Underlined text</u>
<mark></mark>
Highlights text like a yellow highlighter marker.
Example: <mark>Highlighted text</mark>
<small></small>
Makes text smaller, often used for fine print or side comments.
Example: <small>Small print text</small>
<sub></sub> and <sup></sup>
Creates subscript (below baseline) and superscript (above baseline) text.
Examples:
H<sub>2</sub>O (H‚ÇÇO)
E=mc<sup>2</sup> (E=mc¬≤)

üîó Link and Navigation Tags

<a></a>
Creates links to other pages, files, or sections. The "href" attribute specifies where to go.
Examples:
<a href="https://google.com">Visit Google</a>
<a href="about.html">About Page</a>
<a href="#section1">Go to Section 1</a>

üñºÔ∏è Media Tags

<img>
Displays an image. Self-closing tag. Always include "alt" text for accessibility!
Example: <img src="photo.jpg" alt="A beautiful sunset">
<video></video>
Embeds video content. Include "controls" to show play/pause buttons.
Example: <video src="movie.mp4" controls></video>
<audio></audio>
Embeds audio content. Include "controls" to show play/pause buttons.
Example: <audio src="music.mp3" controls></audio>

üìã List Tags

<ul></ul>
Creates an unordered (bulleted) list. Contains <li> items.
Example:
<ul>
  <li>First item</li>
  <li>Second item</li>
</ul>
<ol></ol>
Creates an ordered (numbered) list. Contains <li> items.
Example:
<ol>
  <li>Step one</li>
  <li>Step two</li>
</ol>
<li></li>
Creates a list item. Must be inside <ul> or <ol> tags.
Example: <li>List item content</li>

üìä Table Tags

<table></table>
Creates a table to display data in rows and columns.
Example: <table>...</table>
<tr></tr>
Creates a table row. Goes inside <table>.
Example: <tr><td>Cell 1</td><td>Cell 2</td></tr>
<td></td>
Creates a regular table cell (table data). Goes inside <tr>.
Example: <td>Cell content</td>
<th></th>
Creates a table header cell. Usually bold and centered. Goes inside <tr>.
Example: <th>Column Header</th>

üìù Form Tags

<form></form>
Creates a form for user input. Contains input fields, buttons, etc.
Example: <form action="submit.php" method="post">...</form>
<input>
Creates various input fields. Self-closing tag. Type determines what kind of input.
Examples:
<input type="text" name="username"> (text box)
<input type="password" name="pass"> (password field)
<input type="email" name="email"> (email field)
<input type="submit" value="Submit"> (submit button)
<textarea></textarea>
Creates a multi-line text input area.
Example: <textarea name="message" rows="4" cols="50"></textarea>
<button></button>
Creates a clickable button.
Example: <button type="submit">Click Me!</button>
<select></select>
Creates a dropdown menu. Contains <option> elements.
Example:
<select name="color">
  <option value="red">Red</option>
  <option value="blue">Blue</option>
</select>
<option></option>
Creates an option in a dropdown menu. Goes inside <select>.
Example: <option value="apple">Apple</option>
<label></label>
Creates a label for form elements. Improves accessibility.
Example: <label for="username">Username:</label>

üèóÔ∏è Semantic Structure Tags

<header></header>
Defines the header section of a page or section. Usually contains navigation, logo, etc.
Example: <header><h1>Site Title</h1><nav>...</nav></header>
<nav></nav>
Contains navigation links (menu items).
Example: <nav><a href="home.html">Home</a><a href="about.html">About</a></nav>
<main></main>
Contains the main content of the page. Should only be used once per page.
Example: <main><h1>Page Title</h1><p>Main content...</p></main>
<section></section>
Defines a section of content, usually with its own heading.
Example: <section><h2>About Us</h2><p>Content...</p></section>
<article></article>
Contains independent, self-contained content like blog posts or news articles.
Example: <article><h2>Blog Post Title</h2><p>Post content...</p></article>
<aside></aside>
Contains content that's related but separate from main content (sidebars, ads, etc.).
Example: <aside><h3>Related Links</h3><ul>...</ul></aside>
<footer></footer>
Defines the footer section, usually containing copyright, contact info, etc.
Example: <footer><p>© 2024 My Website</p></footer>

üì¶ Generic Container Tags

<div></div>
A generic container for grouping content. Has no special meaning - use for styling/layout.
Example: <div class="container"><p>Content here</p></div>
<span></span>
A generic inline container for small portions of text. Use for styling individual words/phrases.
Example: <p>The <span class="highlight">important word</span> stands out.</p>

üíª Code and Technical Tags

<code></code>
Displays inline code snippets in a monospace font.
Example: <p>Use the <code>console.log()</code> function.</p>
<pre></pre>
Preserves whitespace and formatting. Great for code blocks.
Example:
<pre>
function hello() {
  console.log("Hello!");
}
</pre>

üé® Style and Script Tags

<style></style>
Contains CSS styles for the page. Usually goes in the <head> section.
Example: <style>h1 { color: blue; }</style>
<script></script>
Contains or links to JavaScript code.
Examples:
<script>alert("Hello!");</script>
<script src="script.js"></script>
<link>
Links to external resources like CSS files. Self-closing tag. Goes in <head>.
Example: <link rel="stylesheet" href="styles.css">

üî§ Special Characters

Examples:
&lt; displays < (less than)
&gt; displays > (greater than)
&amp; displays & (ampersand)
&quot; displays " (quote)
&copy; displays ¬© (copyright)
&nbsp; creates a non-breaking space
üéØ Quick Tips for Beginners:
  • Always close your tags! Every <tag> needs a </tag> (except self-closing ones)
  • Nest properly: <p><strong>text</strong></p> ‚úì not <p><strong>text</p></strong> ‚úó
  • Use semantic tags: Use <header>, <nav>, <main> instead of just <div> everywhere
  • Include alt text on images for accessibility
  • Validate your HTML using online validators to catch errors
  • Practice! The best way to learn is by building actual web pages
