üìö 1. Basic Structure of a Java Program

Every Java program needs a class and a main method to run:

public class MyFirstProgram {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
Note: The file name must match the class name (e.g., MyFirstProgram.java)

üî§ 2. Variables and Data Types

Variables store data. Java has different types for different kinds of data:

Primitive Data Types:

Type What it stores Example
int Whole numbers int age = 25;
double Decimal numbers double price = 19.99;
float Decimal (less precise) float temp = 98.6f;
char Single character char grade = 'A';
boolean True or False boolean isOpen = true;
String Text (multiple characters) String name = "John";

Declaring Variables:

int myNumber = 10;        // Integer
double myDecimal = 5.5;   // Decimal
String myText = "Hello";  // Text
boolean myBoolean = true; // True/False

‚ûï 3. Operators

Arithmetic Operators:

int a = 10, b = 3;
int sum = a + b;      // Addition: 13
int diff = a - b;     // Subtraction: 7
int product = a * b;  // Multiplication: 30
int quotient = a / b; // Division: 3
int remainder = a % b;// Modulus (remainder): 1

Comparison Operators:

a == b  // Equal to
a != b  // Not equal to
a > b   // Greater than
a < b   // Less than
a >= b  // Greater than or equal to
a <= b  // Less than or equal to

Logical Operators:

&&  // AND (both must be true)
||  // OR (at least one must be true)
!   // NOT (reverses true/false)

üîÄ 4. Control Flow Statements

If-Else Statements:

Make decisions in your code:

int age = 18;

if (age >= 18) {
    System.out.println("You are an adult");
} else if (age >= 13) {
    System.out.println("You are a teenager");
} else {
    System.out.println("You are a child");
}

Switch Statement:

Choose between multiple options:

int day = 3;

switch (day) {
    case 1:
        System.out.println("Monday");
        break;
    case 2:
        System.out.println("Tuesday");
        break;
    case 3:
        System.out.println("Wednesday");
        break;
    default:
        System.out.println("Other day");
}

üîÅ 5. Loops

Repeat code multiple times:

For Loop:

When you know how many times to loop:

for (int i = 0; i < 5; i++) {
    System.out.println("Count: " + i);
}
// Prints: Count: 0, Count: 1, Count: 2, Count: 3, Count: 4

While Loop:

Loop while a condition is true:

int i = 0;
while (i < 5) {
    System.out.println("Count: " + i);
    i++;
}

Do-While Loop:

Runs at least once, then checks condition:

int i = 0;
do {
    System.out.println("Count: " + i);
    i++;
} while (i < 5);

üì¶ 6. Arrays

Store multiple values in one variable:

Creating Arrays:

// Method 1: Declare size first
int[] numbers = new int[5];
numbers[0] = 10;
numbers[1] = 20;

// Method 2: Initialize with values
String[] fruits = {"Apple", "Banana", "Orange"};

// Accessing elements
System.out.println(fruits[0]); // Prints: Apple

// Array length
System.out.println(fruits.length); // Prints: 3

Looping Through Arrays:

String[] colors = {"Red", "Green", "Blue"};

// For loop
for (int i = 0; i < colors.length; i++) {
    System.out.println(colors[i]);
}

// Enhanced for loop (for-each)
for (String color : colors) {
    System.out.println(color);
}

üîß 7. Methods (Functions)

Reusable blocks of code:

// Method that doesn't return anything
public static void greet() {
    System.out.println("Hello!");
}

// Method that returns a value
public static int add(int a, int b) {
    return a + b;
}

// Method with parameters
public static void sayHello(String name) {
    System.out.println("Hello, " + name + "!");
}

// Using methods
greet();                    // Prints: Hello!
int sum = add(5, 3);        // sum = 8
sayHello("Alice");          // Prints: Hello, Alice!
Note: static means the method belongs to the class, not an object. void means it doesn't return a value.

üèóÔ∏è 8. Classes and Objects

Classes are blueprints; objects are instances of classes:

Creating a Class:

public class Car {
    // Properties (attributes)
    String brand;
    String color;
    int year;
    
    // Constructor (creates objects)
    public Car(String b, String c, int y) {
        brand = b;
        color = c;
        year = y;
    }
    
    // Method
    public void drive() {
        System.out.println("The " + color + " " + brand + " is driving!");
    }
}

Creating and Using Objects:

// Create objects
Car myCar = new Car("Toyota", "Red", 2020);
Car yourCar = new Car("Honda", "Blue", 2021);

// Access properties
System.out.println(myCar.brand);  // Prints: Toyota

// Call methods
myCar.drive();  // Prints: The Red Toyota is driving!

üì• 9. Input and Output

Output (Printing):

System.out.println("This prints with a new line");
System.out.print("This prints without a new line");
System.out.printf("Formatted: %d %s", 5, "text");

Input (Reading from user):

import java.util.Scanner;

Scanner input = new Scanner(System.in);

System.out.print("Enter your name: ");
String name = input.nextLine();

System.out.print("Enter your age: ");
int age = input.nextInt();

System.out.println("Hello " + name + ", you are " + age);

input.close();  // Always close the Scanner!

üéØ 10. Common String Methods

String text = "Hello World";

text.length()           // Returns: 11
text.toUpperCase()      // Returns: "HELLO WORLD"
text.toLowerCase()      // Returns: "hello world"
text.charAt(0)          // Returns: 'H'
text.substring(0, 5)    // Returns: "Hello"
text.contains("World")  // Returns: true
text.replace("World", "Java")  // Returns: "Hello Java"
text.trim()             // Removes whitespace from ends
text.equals("Hello")    // Compares strings (returns false)
