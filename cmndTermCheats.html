â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â• NAVIGATION COMMANDS â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•
pwd
Print Working Directory - Shows where you currently are
$ pwd
/Users/yourname/Documents
Windows equivalent: cd (with no arguments)
ls
List - Shows all files and folders in current directory
$ ls
file1.txt file2.txt my-folder
Windows equivalent: dir
Tip: Use "ls -la" to see hidden files and details
cd
Change Directory - Move to a different folder
$ cd Documents
$ cd /Users/yourname/Desktop
$ cd .. (go up one level)
$ cd ~ (go to home directory)
Works on both Windows and Mac/Linux!
clear
Clears the terminal screen
$ clear
Windows equivalent: cls
â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â• FILE OPERATIONS â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•
touch
Creates a new empty file
$ touch newfile.txt
$ touch index.html style.css
Windows equivalent: type nul > filename.txt
cat
Shows the contents of a file
$ cat myfile.txt
Windows equivalent: type filename.txt
cp
Copy - Copies files or folders
$ cp file.txt copy.txt (copy a file)
$ cp -r folder1 folder2 (copy entire folder)
Windows equivalent: copy (for files), xcopy (for folders)
mv
Move - Moves OR renames files/folders
$ mv oldname.txt newname.txt (rename)
$ mv file.txt Documents/ (move to Documents)
Windows equivalent: move (for moving), ren (for renaming)
rm
Remove - Deletes files (BE CAREFUL! This is permanent!)
$ rm file.txt (delete a file)
$ rm -r folder (delete entire folder)
Windows equivalent: del (files), rmdir /s (folders)
âš ï¸ WARNING: Deleted files don't go to trash - they're gone forever!
â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â• FOLDER OPERATIONS â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•
mkdir
Make Directory - Creates a new folder
$ mkdir my-project
$ mkdir folder1 folder2 folder3 (create multiple)
Works on both Windows and Mac/Linux!
rmdir
Remove Directory - Deletes an EMPTY folder
$ rmdir old-folder
Use "rm -r" on Mac/Linux or "rmdir /s" on Windows to delete non-empty folders
â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â• VIEWING & SEARCHING â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•
less
View file contents page by page (good for long files)
$ less longfile.txt
Press Q to quit, Space to scroll down, arrows to navigate
Windows equivalent: more
head
Shows the first 10 lines of a file
$ head file.txt
$ head -n 20 file.txt (show first 20 lines)
Mac/Linux only
tail
Shows the last 10 lines of a file
$ tail file.txt
$ tail -f logfile.txt (follow file as it updates)
Mac/Linux only
grep
Searches for text within files
$ grep "search term" file.txt
$ grep -r "function" . (search all files in current folder)
Windows equivalent: findstr
find
Finds files and folders by name
$ find . -name "*.txt" (find all .txt files)
$ find . -type d -name "project" (find folders named project)
Windows equivalent: dir /s filename
â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â• PERMISSIONS (Mac/Linux) â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•
chmod
Change file permissions (who can read/write/execute)
$ chmod +x script.sh (make file executable)
$ chmod 755 file.txt (set specific permissions)
Mac/Linux only
sudo
Run command as administrator (superuser)
$ sudo apt-get install package
Mac/Linux only - You'll be asked for your password
Windows equivalent: Run as Administrator

date
Shows current date and time
$ date
Works on both Windows and Mac/Linux!

Mac/Linux - shows recent commands you've typed
Windows equivalent: doskey /history
echo
Prints text to the screen

Works on both Windows and Mac/Linux!

â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â• SHORTCUTS & TRICKS â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•
Tab Key
Auto-completes file/folder names
Start typing and press Tab - it'll finish the name for you!
Up/Down Arrows
Cycle through previous commands
Press Up to see your last command, keep pressing to go back further
Ctrl + C
Stops/cancels the current running command
Use this when something is stuck or taking too long
Ctrl + A
Moves cursor to beginning of line
Ctrl + E
Moves cursor to end of line
Ctrl + L
Clears the screen (like the "clear" command)
Ctrl + D
Exits the terminal
â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â• COMBINING COMMANDS â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•
| (pipe)
Sends output of one command to another
$ ls | grep "test" (list files, then search for "test")
> (redirect)
Saves command output to a file
$ ls > filelist.txt (save file list to a text file)
This OVERWRITES the file!
>> (append)
Adds command output to end of a file
$ echo "new line" >> file.txt (add text to end of file)
&& (and)
Runs second command only if first succeeds
$ mkdir project && cd project (create folder and enter it)
