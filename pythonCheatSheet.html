üìù 1. Variables & Data Types

Variables: Containers that store data values. Think of them as labeled boxes.

# Variables don't need declaration, just assign a value name = "Alice" # String (text) age = 25 # Integer (whole number) height = 5.6 # Float (decimal number) is_student = True # Boolean (True/False)
üí° Note: Variable names should be lowercase with underscores (snake_case). They can't start with numbers!

üî¢ 2. Basic Data Types

Type Description Example
str String - text data "Hello World"
int Integer - whole numbers 42
float Float - decimal numbers 3.14
bool Boolean - True or False True
list Ordered, changeable collection [1, 2, 3]
tuple Ordered, unchangeable collection (1, 2, 3)
dict Key-value pairs {"name": "Bob"}

‚ûï 3. Operators

Arithmetic Operators:

+ (addition) # 5 + 3 = 8 - (subtraction) # 5 - 3 = 2 * (multiplication) # 5 * 3 = 15 / (division) # 5 / 2 = 2.5 // (floor division) # 5 // 2 = 2 (rounds down) % (modulus) # 5 % 2 = 1 (remainder) ** (exponent) # 5 ** 2 = 25 (5 squared)

Comparison Operators: (Return True or False)

== (equal to) # 5 == 5 is True != (not equal to) # 5 != 3 is True > (greater than) # 5 > 3 is True < (less than) # 5 < 3 is False >= (greater or equal) # 5 >= 5 is True <= (less or equal) # 5 <= 3 is False

Logical Operators:

and # Both conditions must be True or # At least one condition must be True not # Reverses the result (True becomes False)

üìã 4. Lists

Lists: Ordered collections that can hold multiple items. They're changeable!

# Creating a list fruits = ["apple", "banana", "cherry"] # Accessing items (indexing starts at 0) first_fruit = fruits[0] # "apple" last_fruit = fruits[-1] # "cherry" (negative = from end) # Common list methods fruits.append("orange") # Add to end fruits.insert(1, "mango") # Insert at position 1 fruits.remove("banana") # Remove specific item fruits.pop() # Remove last item fruits.sort() # Sort alphabetically len(fruits) # Get length of list

üìñ 5. Dictionaries

Dictionaries: Store data in key-value pairs. Like a real dictionary where you look up a word (key) to find its definition (value).

# Creating a dictionary person = { "name": "Alice", "age": 25, "city": "New York" } # Accessing values print(person["name"]) # "Alice" print(person.get("age")) # 25 # Adding/updating person["job"] = "Engineer" # Add new key-value person["age"] = 26 # Update existing value # Common methods person.keys() # Get all keys person.values() # Get all values person.items() # Get all key-value pairs

üîÅ 6. If Statements (Conditionals)

If statements: Make decisions in your code. Execute different code based on conditions.

age = 18 if age >= 18: print("You are an adult") elif age >= 13: print("You are a teenager") else: print("You are a child")
üí° Note: Python uses indentation (4 spaces or 1 tab) to define code blocks. This is REQUIRED!

üîÑ 7. Loops

For Loops: Repeat code for each item in a sequence.

# Loop through a list fruits = ["apple", "banana", "cherry"] for fruit in fruits: print(fruit) # Loop through a range of numbers for i in range(5): # 0, 1, 2, 3, 4 print(i) for i in range(2, 6): # 2, 3, 4, 5 print(i)

While Loops: Repeat code while a condition is True.

count = 0 while count < 5: print(count) count += 1 # Same as: count = count + 1
üéØ Example: Break and continue
for i in range(10): if i == 3: continue # Skip this iteration if i == 7: break # Exit the loop completely print(i)

üõ†Ô∏è 8. Functions

Functions: Reusable blocks of code that perform a specific task.

# Defining a function def greet(name): return f"Hello, {name}!" # Calling a function message = greet("Alice") print(message) # Output: Hello, Alice! # Function with multiple parameters def add_numbers(a, b): return a + b result = add_numbers(5, 3) # 8 # Function with default parameter def greet_with_title(name, title="Mr."): return f"Hello, {title} {name}"

üì•üì§ 9. Input & Output

Getting user input and displaying output:

# Output (printing to screen) print("Hello, World!") print("Value:", 42) print(f"My name is {name} and I am {age}") # f-string # Input (getting data from user) name = input("What is your name? ") age = int(input("What is your age? ")) # Convert to integer
üí° Note: input() always returns a string. Use int() or float() to convert to numbers!

üìù 10. String Methods

text = "Hello World" text.upper() # "HELLO WORLD" text.lower() # "hello world" text.strip() # Remove whitespace from ends text.replace("H", "J") # "Jello World" text.split() # ["Hello", "World"] text.startswith("He") # True text.endswith("ld") # True len(text) # 11 (length)

üí¨ 12. Comments

Comments: Notes in your code that Python ignores. Use them to explain what your code does!

# This is a single-line comment """ This is a multi-line comment. You can write multiple lines here. Useful for longer explanations. """ x = 5 # You can also add comments after code

‚ö†Ô∏è 13. Try-Except (Error Handling)

Try-Except: Handle errors gracefully without crashing your program.

try: number = int(input("Enter a number: ")) result = 10 / number print(result) except ValueError: print("That's not a valid number!") except ZeroDivisionError: print("You can't divide by zero!") except: print("Something went wrong!") finally: print("This runs no matter what")

üì¶ 14. Importing Modules

Modules: Pre-written code you can use in your programs.

# Import entire module import math print(math.sqrt(16)) # 4.0 # Import specific functions from math import sqrt, pi print(sqrt(16)) # 4.0 print(pi) # 3.14159... # Import with alias import random as rand print(rand.randint(1, 10)) # Random number 1-10

üéØ 15. List Comprehensions

List Comprehensions: A shorter way to create lists based on existing lists.

# Traditional way squares = [] for i in range(10): squares.append(i ** 2) # List comprehension (same result, shorter!) squares = [i ** 2 for i in range(10)] # With condition even_squares = [i ** 2 for i in range(10) if i % 2 == 0]
