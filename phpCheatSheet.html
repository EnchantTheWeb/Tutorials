1. PHP Basics

What is PHP? PHP (Hypertext Preprocessor) is a server-side scripting language used to create dynamic web pages. It runs on the server and generates HTML that gets sent to the browser.

PHP Tags

<?php
// Your PHP code goes here
echo "Hello, World!";
?>
Note: All PHP code must be written between <?php and ?> tags. If your file contains only PHP code, you can omit the closing ?> tag.

2. Comments

Comments help you document your code. They are ignored by PHP.

<?php
// This is a single-line comment

# This is also a single-line comment

/* This is a 
   multi-line comment
   spanning multiple lines */

echo "Comments won't show in the output";
?>

3. Variables

Variables store data. In PHP, variables start with a $ sign. 

<?php
$name = "John";           // String (text)
$age = 25;                // Integer (whole number)
$price = 19.99;           // Float (decimal number)
$isStudent = true;        // Boolean (true/false)

echo $name;               // Outputs: John
?>
Tip: Variable names are case-sensitive! $name and $Name are different variables.

4. Echo and Print

These output data to the screen.

<?php
echo "Hello World";       // Faster, can output multiple values
print "Hello World";      // Slower, returns 1

echo "Hi", " ", "there";  // Multiple values
echo "The answer is " . $age;  // Concatenation with dot
?>

5. Data Types

Type Description Example
String Text characters "Hello" or 'Hello'
Integer Whole numbers 42, -10
Float Decimal numbers 3.14, -0.5
Boolean True or false true, false
Array Multiple values array(1, 2, 3)
NULL No value null

6. Operators

Arithmetic Operators

<?php
$a = 10;
$b = 3;

echo $a + $b;  // Addition: 13
echo $a - $b;  // Subtraction: 7
echo $a * $b;  // Multiplication: 30
echo $a / $b;  // Division: 3.33...
echo $a % $b;  // Modulus (remainder): 1
echo $a ** $b; // Exponentiation: 1000
?>

Comparison Operators

<?php
$a == $b   // Equal to
$a === $b  // Identical (equal value AND same type)
$a != $b   // Not equal
$a !== $b  // Not identical
$a < $b    // Less than
$a > $b    // Greater than
$a <= $b   // Less than or equal to
$a >= $b   // Greater than or equal to
?>

Logical Operators

<?php
$a and $b  // True if both are true
$a && $b   // Same as 'and' but higher precedence
$a or $b   // True if either is true
$a || $b   // Same as 'or' but higher precedence
!$a        // True if $a is false (NOT)
?>

7. Conditional Statements

If Statement

<?php
$age = 18;

if ($age >= 18) {
    echo "You are an adult";
} elseif ($age >= 13) {
    echo "You are a teenager";
} else {
    echo "You are a child";
}
?>

Switch Statement

<?php
$day = "Monday";

switch ($day) {
    case "Monday":
        echo "Start of the week";
        break;
    case "Friday":
        echo "Almost weekend!";
        break;
    default:
        echo "It's just another day";
}
?>
Note: Don't forget break in switch statements or it will continue to the next case!

8. Loops

While Loop

<?php
$i = 1;
while ($i <= 5) {
    echo $i;
    $i++;  // Increment
}
// Outputs: 12345
?>

For Loop

<?php
for ($i = 1; $i <= 5; $i++) {
    echo $i;
}
// Outputs: 12345
?>

Foreach Loop (for arrays)

<?php
$colors = array("red", "green", "blue");

foreach ($colors as $color) {
    echo $color . " ";
}
// Outputs: red green blue
?>

9. Arrays

Indexed Arrays

<?php
$fruits = array("Apple", "Banana", "Orange");
// Or modern syntax:
$fruits = ["Apple", "Banana", "Orange"];

echo $fruits[0];  // Outputs: Apple (arrays start at 0!)
echo $fruits[1];  // Outputs: Banana
?>

Associative Arrays (with keys)

<?php
$person = array(
    "name" => "John",
    "age" => 25,
    "city" => "New York"
);

echo $person["name"];  // Outputs: John
echo $person["age"];   // Outputs: 25
?>

Useful Array Functions

<?php
count($fruits);           // Number of elements
array_push($fruits, "Grape");  // Add to end
array_pop($fruits);       // Remove from end
in_array("Apple", $fruits);  // Check if exists
sort($fruits);            // Sort alphabetically
?>

10. Functions

Functions are reusable blocks of code.

<?php
// Define a function
function sayHello($name) {
    echo "Hello, " . $name;
}

// Call the function
sayHello("Alice");  // Outputs: Hello, Alice

// Function with return value
function add($a, $b) {
    return $a + $b;
}

$result = add(5, 3);  // $result is now 8
?>

11. String Functions

<?php
$text = "Hello World";

strlen($text);              // Length: 11
strtoupper($text);          // HELLO WORLD
strtolower($text);          // hello world
str_replace("World", "PHP", $text);  // Hello PHP
substr($text, 0, 5);        // Hello
trim("  text  ");           // Removes whitespace
explode(" ", $text);        // Splits into array
?>

12. Superglobals

Special variables available everywhere in your script.

<?php
$_GET      // Data from URL parameters
$_POST     // Data from forms (POST method)
$_REQUEST  // Data from both GET and POST
$_SESSION  // Session variables (stored on server)
$_COOKIE   // Cookie data
$_SERVER   // Server information
$_FILES    // Uploaded files

// Example:
// URL: page.php?name=John
echo $_GET['name'];  // Outputs: John
?>

13. Form Handling

<!-- HTML Form -->
<form method="POST" action="process.php">
    <input type="text" name="username">
    <input type="submit" value="Send">
</form>

<?php
// process.php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $username = $_POST['username'];
    echo "Hello, " . $username;
}
?>

14. File Handling

<?php
// Read file
$content = file_get_contents("file.txt");

// Write to file
file_put_contents("file.txt", "Hello World");

// Open file for reading
$file = fopen("file.txt", "r");
$content = fread($file, filesize("file.txt"));
fclose($file);

// Open file for writing
$file = fopen("file.txt", "w");
fwrite($file, "New content");
fclose($file);
?>

15. Include/Require

Insert code from another file.

<?php
include 'header.php';     // Includes file, continues if not found
require 'header.php';     // Includes file, stops if not found

include_once 'header.php';  // Includes only once
require_once 'header.php';  // Requires only once
?>

16. Database (MySQL)

<?php
// Connect to database
$conn = mysqli_connect("localhost", "username", "password", "database");

// Check connection
if (!$conn) {
    die("Connection failed: " . mysqli_connect_error());
}

// Query database
$sql = "SELECT * FROM users";
$result = mysqli_query($conn, $sql);

// Fetch data
while ($row = mysqli_fetch_assoc($result)) {
    echo $row['name'];
}

// Close connection
mysqli_close($conn);
?>

Function What it does

print_r() Display array/object in readable form
date() Format date/time
rand() Generate random number

18. Error Handling

<?php
// Try-Catch block
try {
    // Code that might cause an error
    $result = 10 / 0;
} catch (Exception $e) {
    echo "Error: " . $e->getMessage();
}

// Display errors (for development only!)
ini_set('display_errors', 1);
error_reporting(E_ALL);
?>

üéØ Quick Tips for Beginners

  • Always end PHP statements with a semicolon ;
  • Variable names are case-sensitive, function names are not
  • Use var_dump() to debug and see what's in a variable
  • Always validate and sanitize user input for security
  • PHP files usually have a .php extension
  • You can mix HTML and PHP in the same file
  • Use // for comments to explain your code
